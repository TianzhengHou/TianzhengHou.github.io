About the Project

This project demonstrates my learning and application of basic Artificial Intelligence concepts using Python, with a particular focus on leveraging NumPy for building various machine learning models. Each script is designed to explore different algorithms and methods covered during my coursework at Dalhousie University.

Project Highlights

Below are some key projects that highlight different aspects of my work in Artificial Intelligence and Python programming:

  • Clustering with Python

    In this project, I implemented various clustering algorithms such as K-Means and hierarchical clustering to analyze and segment data points. The goal was to learn how unsupervised learning can be used to discover patterns in data without prior labels.

  • K-Nearest Neighbors (KNN) Algorithm

    This project involved building a K-Nearest Neighbors classifier from scratch. The KNN algorithm is a simple, yet powerful tool for classification, particularly useful in situations where the decision boundary is non-linear.

  • Linear Regression Model

    I developed a linear regression model to predict continuous outcomes based on input features. This project helped solidify my understanding of regression analysis and the underlying mathematics of gradient descent.

  • Stochastic Neural Network

    This neural network model incorporates stochastic elements to better capture the randomness inherent in certain types of data. The stochastic approach is particularly useful in modeling non-deterministic processes where outcomes can vary.

  • Tic-Tac-Toe with Reinforcement Learning

    In this reinforcement learning project, the model learns to play Tic-Tac-Toe by exploring different strategies and learning from outcomes. The project showcases how reinforcement learning can be used to develop intelligent agents that improve through experience.
