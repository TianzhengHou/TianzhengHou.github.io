About the Repository

This repository showcases my work on Information Retrieval algorithms, which I studied at Dalhousie University. It includes various scripts that implement different retrieval methods, focusing on optimizing search efficiency and accuracy.

The repository reflects my interest in how information can be systematically indexed, retrieved, and processed to deliver relevant search results. You can learn more about my background and interests on the About Me page.

Repository Files

The repository contains several Python scripts, each focusing on a different aspect of Information Retrieval:

  • BooleanSearch.py

    This script implements a Boolean search algorithm using a faster term index matrix. It provides an efficient way to retrieve documents that match a specific set of keywords.

  • DataCleaning.py

    The DataCleaning script preprocesses text data by removing noise, punctuation, and stop words, which helps improve the accuracy of retrieval algorithms.

  • InvertedIndexSearch.py

    This script develops an inverted index-based search engine, allowing for quick lookup of documents containing specific terms. The inverted index is a fundamental data structure for text retrieval systems.

  • TermDocumentIncidenceMatrix.py

    It computes a term-document incidence matrix, which represents the presence or absence of terms across a set of documents. This matrix is a key component in building search engines.

  • Text_to_Image_search_Transformer.py

    This script utilizes OpenAI's CLIP model to implement a text-to-image search transformer, allowing for searching images based on textual descriptions.
